# CIB - Mosaic Octopus

Breakout board for Pico Technologies Ltd ADC20/ADC24 (https://www.picotech.com/accessories/data-logger-terminal-boards/adc-20-24-terminal-board)

> The ADC-20/ADC-24 Terminal Board is designed for use with the 8-channel ADC-20 and 16-channel ADC-24 High-Resolution Data Loggers. For simple applications, you can simply connect sensor wires to the screw terminals on the terminal board without the need for soldering. For more advanced applications, the terminal board enables you to design and build sensor circuits that condition measurements for the data loggers to process. The board has empty locations for extra components.
